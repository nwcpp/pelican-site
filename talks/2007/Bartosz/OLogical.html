Logical Contents

  • Simpler than in WSTM
    • No many to one address/orec mapping
    • No need for version numbers
  • If header points to data, that's the logical value
  • If header points to transaction
    • SUCCEEDED: the new value (the shadow copy)
    • FAILED: the old value
    • READ-CHECK: undefined (should help the owning transaction)
