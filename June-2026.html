Title

Wait-Free Programming From Scratch

Abstract

Wait-free algorithms are a fundamental part of many real-time software systems. Using an example from real-time audio software, this talk explores several generic strategies for wait-free communication between threads. After evaluating the trade-offs of the generic solutions, the talk walks through a “custom” solution built from first principles.

Speaker Bio

Jatin Chowdhury is a Ph.D. student in electrical engineering and computer science at MIT, specializing in music technology. He has worked as an audio signal processing engineer and software developer. Jatin currently lives in Renton, WA, and runs Chowdhury DSP, a boutique audio software company.
